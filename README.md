# 贵州鹦鹉螺文化网站

贵州鹦鹉螺文化官方网站，提供公司介绍、业务展示、联系信息和在线留言功能。

## 已实现功能

- 响应式企业官网首页，适配电脑和手机访问
- 公司地址与“高德地图导航”入口
- 公司电话一键拨号功能
- 联系留言表单
- 图片验证码和简单的防机器人校验
- 留言数据保存到本地 JSON 数据库
- 管理员登录后台查看留言
- 留言接口限流、登录失败限制和表单蜜罐校验

## 本地运行

项目使用 Node.js 内置模块，无需额外安装业务依赖：

```bash
node server.js
```

默认访问地址：`http://127.0.0.1:8765/`

后台地址：`http://127.0.0.1:8765/admin.html`

## 配置项

生产环境建议通过环境变量设置：

- `PORT`：网站端口，默认 `8765`
- `ADMIN_USER`：后台用户名
- `ADMIN_PASSWORD`：后台密码
- `CAPTCHA_SECRET`：验证码签名密钥

留言数据位于 `data/messages.json`。该文件不提交到 GitHub，以避免把用户留言带入代码仓库。

## GitHub 部署

服务器可以使用仓库中的 `deploy-from-github.sh` 拉取最新代码并重启 PM2 服务。部署脚本需要提前设置 `ADMIN_PASSWORD` 和 `CAPTCHA_SECRET` 环境变量，并由 Nginx 将公网流量转发到 Node.js 服务。

## 注意事项

- 不要把服务器密码、后台密码或验证码密钥提交到 GitHub。
- 生产环境建议绑定域名并启用 HTTPS。
- 使用高德地图导航时，请根据实际高德地图链接更新首页地址入口。
